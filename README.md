# GenX Lead Intelligence Suite v4.5 (OP Edition)

A professional, high-performance SaaS dashboard powered by **OpenRouter Gemini 2.0 Flash**. This edition is optimized for 100% search-verified executive intelligence.

![GenX Pro](https://img.shields.io/badge/Status-Deployment--Ready-indigo?style=for-the-badge)
![Engine](https://img.shields.io/badge/Engine-OpenRouter--OP-orange?style=for-the-badge)

## 🚀 Core Features

- **OpenRouter Intelligence**: Powered by `google/gemini-2.0-flash-001` via OpenRouter for the most accurate, live-search verified results.
- **Midnight Premium UI**: A beautiful, responsive "Midnight Indigo" dashboard with glassmorphism effects and real-time system monitoring.
- **Secure Access System**: Built-in "Peace" login protocol granting 2,000 daily credits.
- **Daily Credit Renewal**: Automated energy handshake that resets user credits every 24 hours.
- **Threaded Background Processing**: Concurrently processes 5 leads at a time for maximum speed.
- **Database Archive**: Save completed batches to a permanent "Archive" tab. Re-download your intelligence reports anytime.

## 🛠️ Installation & Setup

### 1. Clone
```bash
git clone https://github.com/Totobi123/GenX-Lead-Intelligence-OP.git
cd GenX-Lead-Intelligence-OP
```

### 2. Install
```bash
npm install
```

### 3. Run
```bash
npm run dev -- --host
```

## 🛰️ Deployment (Live Server)

```bash
# Start the application permanently
npx pm2 start "npm run dev -- --host" --name "ceo-finder-op"
npx pm2 save
```

## 📖 How to Use

1. **Login**: Use access pass `peace` for 2,000 credits.
2. **Inbound**: Import your leads via the **"Import Leads"** button (Excel, CSV, TXT supported).
3. **Terminal**: Click **"Start Search"** to begin the OpenRouter-powered extraction.
4. **Archive**: Use **"Archive Batch"** to save results to your permanent history.
